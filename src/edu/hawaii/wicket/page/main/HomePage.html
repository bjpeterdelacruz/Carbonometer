Carbonometer
Our goal

This application provides energy-related information for all power sources on the island of Oahu in Hawaii. Our goal is to initiate the connection between the power consumer and the green evolution. Currently, a user can get information about the Oahu grid from five different pages, each of which is described below.

Current Carbon Intensity

This page displays the current carbon intensity level (pounds of carbon dioxide equivalent per megawatt-hour, or lbs CO2 / MWh) of the Oahu grid. The information is updated every minute. Also, this page provides useful tips for consumers on how to wisely manage their power usage.

Grid Information

This page provides the consumer with a list of options to display power-related information on a static line graph.

Grid Visualization

This page provides the consumer with a list of options to display power-related information on a visualized line graph.

Daily Carbon Intensity

This page displays the carbon intensity level of the Oahu grid for a given day from 00:00 to 23:00 hours. If the majority of values for the carbon intensity level for a given day is below the minimum threshold, between the minimum and maximum thresholds, and above the maximum threshold, the stoplight will display a green, yellow, or red light, respectively.

Source Summary

This page displays information about a power source, such as whether it has subsources, its location, its fuel type, etc.
